## Description

This Prework Study Guide was created to help students dip in to basic web development. It contains notes on HTML, CSS, Git, and JavaScript.

## Usage

To use this Prework Study Guide, you can review the notes in each section. For suggestions on what to study first, open the Chrome DevTools by right clicking the webpage and selecting "Inspect". A console panel should open either below or to the side of the webpage in the browser. There you will see a list of topics we learned from the prework along with a suggestion on which topic to study first.
